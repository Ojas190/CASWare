# CASWare
College Administrative System Ware
CAS is an System made to automate the Attendance system using Campus Camera's and providing an Smart, Easy to use Dashboard,  for students and Parents to see what their child does and their attendance being transparent. It also helps in keeping transparency, Accountability and Records as it is an Mini-ERP system capable of using and providing ease to do Task for the Faculty and few features capable to guide Students, while having an self sustained working capability.
